# wayland-root-exec
A simple bash script to run a grafical programm as root in wayland.
# How to use
    wayland-root-exec "command --example"
# Install
    $ sudo cp wayland-root-exec /bin/wayland-root-exec
    $ sudo chmod +x /bin/wayland-root-exec
